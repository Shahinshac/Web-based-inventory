### Database Issues
- Render uses ephemeral filesystem
- Database resets on each deploy
- For persistent data, upgrade to Render PostgreSQL

### App Not Starting
Check Render logs:
1. Go to your service dashboard
2. Click "Logs" tab
3. Look for errors

### Login Issues
- Clear browser cache/cookies
- Try incognito mode
- Check admin login at `/sys-admin`
